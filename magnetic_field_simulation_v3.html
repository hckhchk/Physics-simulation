1제작 배경 Background

2026학년도 과학아카데미 포트폴리오 「직선 도선이 만드는 자기장과 지구자기장 측정」 실험 기반. 학생들이 가상으로 실험 환경을 구성하고 변인을 조작하여 자기장 개념을 체감하도록 설계.

2환경 구성 단계 Setup Phase

  • 드래그 앤 드롭: 5종 도구를 순서대로 배치하며 실험 셋업 학습
  • 전원장치-전류계 인터랙션: 전압 조절 → 옴의 법칙(V=IR) → 전류계 실시간 반응
  • 절차 내면화: 모눈종이→나침반→도선→전원장치→전류계 순서를 자연스럽게 습득

3시뮬레이션 구성 Components

  • 실험 시뮬레이션: I, r, 방향 조작. Top/Side/3D 시점
  • 벡터 합성: B_earth·B_wire 합성 다이어그램
  • 데이터 분석: 1/tanθ vs I 그래프, 선형 회귀로 B_earth 역산

4교육 활용 Pedagogy

  • 사전 학습(Pre-Lab): 환경 구성 연습 및 예상 결과 추론
  • 개념 강화: 오른손 법칙, 벡터 합성 시각적 이해
  • 데이터 리터러시: 노이즈 데이터에서 패턴 추출

5실험 조건 Setup

  • 통제 변인: r = 1.0 cm
  • 조작 변인: I = 0.5 ~ 2.5 A (0.5A 단위)
  • 참값: INTERMAGNET 청양 관측소 약 50.6 μT
  • 안전: 최대 전류 2.5A 제한
